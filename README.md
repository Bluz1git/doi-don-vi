## Bộ Đổi Đơn Vị "Mỹ Tho"

Đây là chiếc tool đổi đơn vị đo lường của Mỹ sang đơn vị Việt Nam.

**Nó làm được gì?**

**Cân nặng:** Đổi từ Pounds (lbs) sang Kg.

**Chiều cao:** Đổi Inch/Ft sang Mét.

Cách dùng: Tải file exe và chạy.
---

## US Unit Converter

This is a tool to convert US measurement units to Vietnamese units.

**What does it do?**

**Weight:** Convert from Pounds (lbs) to Kg.

**Height:** Convert Inch/Ft to Meters.

How to use: Insatll exe file and run it.
